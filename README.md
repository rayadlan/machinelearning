# Fake News Detection App

Versi ini sudah memakai XGBoost asli melalui xgboost.XGBClassifier, sesuai model yang disebut di PPT.

## Isi folder
- app.py
- training_model.py
- requirements.txt
- data/contoh_dataset_50.csv
- models/

## Cara menjalankan
1. pip install -r requirements.txt
2. python training_model.py
3. streamlit run app.py
